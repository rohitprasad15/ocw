# Open Coworker

**An open-source AI coworker that lives on your desktop.** Chat, work in your
codebase, or hand off a task and get a finished deliverable back — from an agent
that runs on your machine and works with the model provider you choose.

**Website:** [www.opencoworker.app](https://www.opencoworker.app)
**Downloads:** [Releases](https://github.com/rohitprasad15/ocw/releases/latest) — macOS (`.dmg`) and Windows installers.

> **Soft launch.** This repo is the temporary home for the Open Coworker website
> and release downloads. The source code is moving to its permanent public home
> shortly — watch this space.

## Install notes

- **macOS:** if Gatekeeper warns that the app is from an unidentified developer,
  right-click the app → **Open** (first launch only).
- **Windows:** SmartScreen may ask for confirmation on first run — choose
  **More info → Run anyway**.
